A personal finance tracker PWA for managing JPY and INR finances. Tracks salary, remittances, recurring bills, debts, and savings goals. Installable on iPhone via Safari.

## Features

- Monthly check-in flow (salary, bills, debts, remittance, spend, goals)
- JPY and INR tracked separately — never combined
- Debt progress with estimated months to clear
- Savings goals with milestone markers
- Trend charts across months
- PWA — installable on iPhone home screen
- All data stored locally in `localStorage` — no backend

## Stack

React + TypeScript + Vite + Tailwind CSS v4 + Vitest
